##  Selected Projects

- [NUnitAPITestsDemo](https://github.com/TodorG1t/NUnitAPITestsDemo)  
  Data-driven NUnit API test suite (.NET 8) demonstrating HTTP request scenarios with reusable HttpClient utilities, structured test data and assertion patterns for stable API testing.

- [NUnitSeleniumUITestsDemo](https://github.com/TodorG1t/NUnitSeleniumUITestsDemo)  
  Demo project scraping Automation QA job positions from dev.bg using NUnit and Selenium WebDriver, with browser configuration, element handling examples and an automated CI/CD pipeline.

- [MonitorSitesCertificateDemo](https://github.com/TodorG1t/MonitorSitesCertificateDemo)  
  Azure Function that monitors SSL certificates for specified domains and tracks their expiration status using Azure Application Insights for telemetry and alerting.

- [PlaywrightVisualInspectionDemo](https://github.com/TodorG1t/PlaywrightVisualInspectionDemo)  
  Visual regression tests using Playwright focused on automated screenshot comparison and visual testing across environments, with baseline management and CI integration.

- [K6Demo](https://github.com/TodorG1t/K6Demo)  
  k6 batch requests demo showcasing concurrent GET/POST/DELETE requests against a mock API with performance thresholds, error tracking and example load scripts.

- [TexasHoldEmDemo](https://github.com/TodorG1t/TexasHoldEmDemo)  
  A fully functional real-time multiplayer Texas Hold'em poker game built entirely through AI prompts (vibe coding), demonstrating modern web development practices such as real-time WebSocket communication, scalable game logic, and responsive UI/UX design.
